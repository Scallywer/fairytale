# Improved DALL-E Prompt Examples with Story Context

## Updated Prompt Template

The prompts now include:
1. **Story Summary** - First 200 characters from the story
2. **Main Characters** - Extracted from "po imenu" patterns
3. **Setting Detection** - Based on keywords (šuma, more, savana, etc.)
4. **Key Visual Elements** - Detected from story content
